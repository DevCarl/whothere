The valid search terms are:

  • Date, which is written in format yy/mm/dd (e.g. 2015/11/25)
  • Week
  • Module, which is the module code in which you are interested (e.g. COMP30250)
  • Room_no, which indicate the name or the code of the room (e.g. B002)

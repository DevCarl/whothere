Instructions for using the API

Our API allows you to query the latest information of the room occupancy in UCD buildings.

Base Url

All our API requests start with the following base:

http://csi420-01-vm1.ucd.ie/api/data

The general query format is the following:

http://csi420-01-vm1.ucd.ie/api/data?request=[search1]&[search1]=[value]

The valid search terms are:

  • Date, which is written in format yy/mm/dd (e.g. 2015/11/25)
  • Week, which it will take a date in the same format as data, but it will return the week of the day selected
  • Module, which is the module code in which you are interested (e.g. COMP30250)
  • Room_no, which indicate the name or the code of the room (e.g. B002)

Example

This is an example on how a simple API query looks like:

http://csi420-01-vm1.ucd.ie/api/data?request=Date&Date=2015/11/11

Complex Url

If you want to make your research more precise you can combine two different requests together as showed below:

http://csi420-01-vm1.ucd.ie/api/data?request=[search1]&[search1]=[value]&request2=[search2]&[search2]=[value]

Example

This is an example on how a complex API query looks like:

http://csi420-01-vm1.ucd.ie/api/data?request=Room_no&Room_no=B003&request2=Date&Date=2015-11-03

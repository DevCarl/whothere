Instructions for using the API

Good Evening Comp Sci Masters students,

TO use our API, the basic request is sent to /api/data

The general extension is as follows ?request=[search1]&[search1]=[value]

You may follow this with &request2=[search2]&[search2]=[value] for more specific results.

An example would be... /api/data?request=Date&Date=2015/11/11

The valid search terms are Date, Week, Module and Room_no.
Week follows the same structure as Date where an individual date is selected.
